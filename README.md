# 🏥 CovidCare – COVID Test & Vaccination Management System

A modern web-based healthcare management platform developed to simplify **COVID-19 test booking**, **vaccination appointments**, and **hospital management** through a secure and user-friendly interface.

CovidCare connects **Patients**, **Hospitals**, and **Administrators** on a single platform, allowing users to manage appointments, vaccination records, COVID-19 test results, and healthcare services efficiently.

---

# 📌 Project Overview

The COVID-19 pandemic created significant challenges for healthcare systems worldwide. CovidCare was developed to provide an efficient online solution for managing COVID-19 testing and vaccination services.

The system allows patients to search approved hospitals, book appointments, monitor vaccination status, and view COVID-19 test results. Hospitals can manage appointment requests and update patient records, while administrators supervise the entire platform through a centralized dashboard.

# 🔄 System Workflow

1. User creates an account.
2. Patient searches approved hospitals.
3. Patient books a COVID Test or Vaccination appointment.
4. Hospital receives appointment request.
5. Hospital approves or rejects the request.
6. Hospital updates test result and vaccination status.
7. Patient tracks appointment and medical records.
8. Administrator monitors the entire system.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/covidcare.git
```

Move into the project folder:

```bash
cd covidcare
```

Install dependencies:

```bash
composer install
```

Copy the environment file:

```bash
cp .env.example .env
```

Generate the application key:

```bash
php artisan key:generate
```

Configure your database inside the **.env** file.

Run migrations:

```bash
php artisan migrate
```

Start the development server:

```bash
php artisan serve
```

Open your browser:

```
http://127.0.0.1:8000
```
